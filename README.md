# OrdenationSelector
Trabalho Prático de Estrutura de Dados 2 da Universidade de São Paulo (USP) do Campus de São Carlos


Exemplos de Uso

Ordenação utilizando Merge Sort:

main merge -aleatorio -tamanho 10000

Ordenação utilizando Heap Sort:

main heap -aleatorio -tamanho 10000

Seleção automática do algoritmo:

main adaptativo -aleatorio -tamanho 10000

Utilizando dados provenientes de arquivo:

main adaptativo -arquivo entrada.txt
